# Measures Converter — Flutter Application

A simple and intuitive Flutter application for converting between **metric and imperial** measurements.  
Users can convert values for **length** and **weight** using a clean mobile interface.

---

## Features

- Convert between metric and imperial units
- Supports:
  - **Length:** meters, kilometers, feet, miles
  - **Weight:** kilograms, grams, pounds, ounces
- Category-based unit selection
- Real-time conversion results
- Clean and responsive UI

---

## Built With

- **Flutter** (UI framework)
- **Dart** (programming language)
- Android Emulator for testing

## Installation & Setup

### Prerequisites

- Flutter SDK installed
- Android Studio or Android Emulator configured

Verify installation:
```bash
flutter doctor
```

---

### Run the App

```bash
git clone https://github.com/YOUR_USERNAME/measures_converter.git
cd measures_converter
flutter pub get
flutter run
```

---

## Usage

1. Enter a numeric value.
2. Select a **Category** (Length or Weight).
3. Choose **From** and **To** units.
4. Press **Convert**.
5. View the converted result instantly.
